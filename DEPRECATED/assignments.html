Travel Planner

Link: https://alecwgreene.github.io/MSU-Project-1/index.html
Team: Alec Greene (Project Lead), Ron Pitts, Alina Gorelik

Description: The main focus of this project was to establish an effective workflow for the team and to practice using the git CLI. As such this project had a tumultuous development cycle, with conflicting ideas about scope and inexperience with the git CLI proving difficult to work around. In the end, the MVP was eventually reached and some additional features were able to be implemented at the 11th hour. The site serves as a 2-in-1 weather forecast and tourist recommendation app. Users can search for places in several pre-selected countries and tourist attractions within 5000m of the location's center will be returned, filtered by the category the user has active.
